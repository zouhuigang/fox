### fox wiki

![images](./image/screen.png)


### 1.下载安装fox,并加入环境变量。


### 2.安装theme

- 全局安装

> 全局安装只需要安装一次

在fox安装目录，打开theme:

	git clone https://github.com/zouhuigang/fox-theme-wiki --depth=1 fox.theme/wiki


- 局部安装:
>局部安装适应需要自定义theme的情况

在需要编辑的目录，新建并打开theme目录:

	git clone https://github.com/zouhuigang/fox-theme-wiki --depth=1 fox.theme/wiki



### 3.当前命令行打开,运行

	fox wiki --theme=wiki
