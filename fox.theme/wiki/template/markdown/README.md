#代码框

>如果你是个程序猿，需要在文章里优雅的引用代码框，在 Markdown 下实现也非常简单，只需要用两个 <code></code>把中间的代码包裹起来。使用 tab 键即可缩进。

    <code>

    include<stdio.h>

    int main(){

    printf("helloword");

    return 0;

    }

	</code>
